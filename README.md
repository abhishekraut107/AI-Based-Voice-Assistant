📌 Overview

The Advanced Python Voice Assistant is an AI-based application that performs tasks using voice commands. It listens to user input, processes it, and responds using speech output. This project aims to provide hands-free interaction with the system for everyday tasks.

🚀 Features
🎙️ Voice command recognition
💻 Open applications (Chrome, VS Code, etc.)
⏰ Tell current time
🌐 Search the web
🔊 Text-to-speech response
🛠️ Technologies Used
     Python
     SpeechRecognition
     pyttsx3
     OS module
     Webbrowser
